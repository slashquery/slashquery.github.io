The configuration file


The configuration file is composed of 3 sections:

  • config
  • routes
  • upstreams
  • plugins

Example:

config:
  socket: /tmp/slashquery.sock

routes:
  get:
    path: /get
    upstream: httpbin
    methods:
      - GET
    plugins:
      - waf

upstream:
  httpbin:
    timeout: 5
    servers:
      - httpbin.org

plugins:
  waf: ["github.com/slashquery-plugins/waf", "waf.WAF"]

The config allow the following options:

  • debug
  • host
  • port
  • request-id
  • resolver
  • socket
